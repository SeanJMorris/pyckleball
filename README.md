# Pyckleball

Pycleball is bot I made to sign up for Pickleball court time with friends.

## Background

This project uses Python, Github Actions, and the Playwright web automation
library to automate signing myself up to play in pick-up Pickleball sessions
posted in my favorite local public online group.

Playwright is most well known for as an end-to-end browser testing framework,
but this was a great use case because there were no APIs I could use to sign up
in the way I needed.
